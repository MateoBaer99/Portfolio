# Portfolio

Personal portfolio website for Mateo Baer - Python Developer & Data Specialist based in Toulouse, France.

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox, Responsive Design)
- Vanilla JavaScript

## Features

- Responsive design with mobile hamburger menu
- Smooth scroll navigation with active section highlighting
- Sections: About, Experience, Education, Certifications, Skills, Languages, Soft Skills

## Project Structure

```
portfolio/
├── index.html       # Main HTML page
├── style.css        # Styles with CSS variables and responsive breakpoints
├── script.js        # Navigation toggle and scroll-based active link logic
└── cv/
    └── cv.md        # CV/Resume in Markdown format
```

## Getting Started

Open `index.html` in a browser, or serve locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```

Then visit `http://localhost:8000`.
